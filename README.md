# Todo Task Card

A testable, accessible Todo / Task Card built with plain HTML, CSS, and JavaScript.

## Features

- Required `data-testid` attributes for automated tests
- Semantic card markup with `article`, `time`, `label`, `button`, and list elements
- Accessible checkbox toggle with visible focus styles
- Responsive layout for mobile, tablet, and desktop screens
- Dynamic time-remaining text that refreshes every 30 seconds

## Deploy Notes

This is a static site. On Netlify, use:

- Build command: leave blank
- Publish directory: `.`
